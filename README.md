# Dashboard de Control de Descuadres - COPEC
**Actividad Práctica de Graduación para Master Business Analytics**

## Descripción del Proyecto
Este repositorio contiene el desarrollo técnico de un Dashboard en Qlik Sense diseñado para el monitoreo y control de descuadres de productos Copec. El objetivo es identificar desviaciones críticas y patrones de comportamiento en los centros operativos.
